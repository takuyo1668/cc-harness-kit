Mac の環境診断とメモリ/ディスク整理を支援するスキルです。

## 手順

### Step 1: 環境診断（自動実行）

以下を並列で取得し、結果を一覧表示する：

- **メモリ**: `sysctl hw.memsize && sysctl hw.ncpu`
- **メモリプレッシャー**: `memory_pressure`
- **仮想メモリ**: `vm_stat | head -20`
- **スワップ**: `sysctl vm.swapusage`
- **ディスク**: `df -h /`
- **メモリ消費上位プロセス**: `ps aux -m | head -25`

### Step 2: 問題の特定と報告

診断結果から以下を判定し、優先度順で報告する：

| チェック項目 | 危険ライン |
|---|---|
| ディスク空き容量 | 5GB 未満 |
| スワップ使用量 | 物理メモリの 2倍超 |
| メモリ空き % | 20% 未満 |
| ゾンビプロセス（古い Claude Code 等） | 不要なプロセスが残留 |

### Step 3: 対策提案

問題ごとに対策を提案する。**以下のルールを厳守すること：**

#### 絶対ルール
- **ファイルの削除は絶対に自動実行しない。** 削除コマンドは提示するだけで、実行は必ず人間が手作業で行う
- **リスクがある操作（プロセス終了、キャッシュ削除等）は事前にリスク説明 + 承認を得てから実行する**
- 削除対象は一覧表にして、サイズ・安全度・判断基準を明示する

#### 提案の順序
1. キャッシュ削除（安全・自動再構築されるもの）
2. 不要プロセスの終了
3. Downloads / 大容量ファイルの整理（人間判断）
4. 根本対策（ハードウェア等）

### Step 4: 実行後の効果確認

対策実行後に再度 `df -h /`、`sysctl vm.swapusage`、`memory_pressure` を実行し、Before/After を表で比較表示する。
