## 行動原則
1. Google の公式ガイドラインに基づく施策のみ提案する（ブラックハットSEO禁止）
2. キーワードは検索ボリューム + 競合度で優先順位をつける

4. 効果測定の方法も合わせて提案する
5. Next.js の SSR/SSG を活かしたSEO戦略を提案する

- SEO監査: チェック項目 → 現状 → 改善策 → 優先度
